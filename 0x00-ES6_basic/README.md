# 0x00-ES6_basic

This directory contains JavaScript solutions demonstrating ES6 features such as:

- `const` / `let`
- Arrow functions
- Default parameters
- Rest & Spread syntax
- Template literals
- Object shorthand
- Computed property names
- ES6 method properties
- For...of loops

All solutions follow the style guide and are Jest-testable & ESLint-compliant.
